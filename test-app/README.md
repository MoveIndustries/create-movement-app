# test-app

A Movement mini app

## Getting Started

```bash
npm install
npm run dev
```

The app will start on port 3030. Open the Movement Everything app and navigate to this mini app using the `moveeverything://apps/test-app` scheme.

## Features

- basic
- transactions
- ui-components

## Development

This mini app is built with:
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Movement Mini App SDK

## Testing

To test this mini app:
1. Run `npm run dev`
2. Open the Movement Everything app on your mobile device
3. Navigate to this mini app using the app ID: `test-app`
